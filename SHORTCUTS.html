AI Assistant (codecompanion — Claude Sonnet)

Space+aaOpen / toggle AI chat buffer
Space+apOpen action palette — all AI actions
Space+aeInline AI edit — type a prompt, Claude edits in place
Space+acAdd visual selection to the open chat

AI Chat — Actions

Enter / Ctrl+sSend message
Ctrl+cClose chat
qStop generation
grRegenerate last response
gxClear chat history
gyYank code from response
gaChange model / adapter
]] / [[Jump to next / previous message
?Show all available keymaps

AI Chat — Context (type # in chat)

#{buffer}Current file contents
#{buffer:filename}A specific open buffer by name
#{buffers}All open buffers
#{selection}Last visual selection
#{diagnostics}LSP errors and warnings in current file
#{diff}Current git diff (staged + unstaged)
#{terminal}Latest output from terminal buffer
#{viewport}Exactly what is visible on screen

AI Chat — Slash Commands (type / in chat)

/bufferPick from open buffers and add to chat
/filePick a file from the project and add to chat
/symbolsAdd a tree-sitter outline of a file (functions, classes)
/fetchFetch a URL and add its content to chat
/compactClear history but keep a summary in context
/helpAdd content from Vim help tags
/nowInsert current date and time
Workflow — chat with full context:
1.Space+aa open chat   2.Type question + #{buffer} to include current file   3.Add more with /file or #{diagnostics} for LSP errors   4.Enter to send · q to stop · gr to regenerate

Go To Line

42G or :42Jump to line 42
ggGo to first line
GGo to last line

LSP — Code Navigation

KHover docs — show documentation popup
gdPeek definition — floating popup, stay in current file
giGo to implementation — jump inside
gDGo to declaration
gfFind all references + definition in a panel
Ctrl+oGo back — return to where you were before the jump
Ctrl+iGo forward in jump history
Space+rnRename symbol — updates every reference
Space+caCode actions — quick fixes, imports, refactors
Inspect without leaving your file:
1.Hover on a function → K to read docs   2.gd to peek in a popup (q closes it, you never moved)   3.gi to jump inside   4.Ctrl+o to jump back

LSP — Diagnostics

Space+dShow diagnostics under cursor
Space+DShow all diagnostics for current line
[d / ]dJump to previous / next diagnostic

pPaste from Vim register
